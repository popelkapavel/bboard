Blackboard 


Blackboard is simple freehand drawing tool onto paper with height and color.
It has several modes for tool height shape and combination.
And has black and white mode.
It can draw various shapes from keyboard.

It is free for any use.

	Pavel Popelka

support: popelkapavel@seznam.cz


Mouse:
Left   - Draw
Right  - Erase
Middle - Pan
Wheel  - Zoom
